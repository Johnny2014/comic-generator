# “4格” 产品策划文档 (Product Concept)

## 1. 产品核心定位
**“4格”** 是一款基于传统中国叙事哲学——**“起、承、转、合”** 的创意短视频生成工具。它将四格漫画的静态美学与现代 AI 大模型技术相结合，通过极简的输入生成具有深度的故事短片。

## 2. 核心叙事架构
产品围绕经典的四阶段叙事模型构建：
- **起 (Introduction)**: 故事背景与开端。
- **承 (Development)**: 情感的延续与铺垫。
- **转 (Transition)**: 意想不到的转折与冲突。
- **合 (Conclusion)**: 最终的伏笔揭晓与总结。

## 3. 功能亮点
- **AI 创意协助**: 利用大模型能力，根据简单的文字描述自动生成契合“起承转合”逻辑的图像素材与分镜。
- **动态视觉引擎**: 采用 Ken Burns 效果让静态面板产生电影级的平移与缩放。
- **多维叙事**: 支持为每一位面板添加独立的**音频说明**与**文案脚本**，全方位塑造故事氛围。
- **成品输出**: 最终一键合成一段 **4 分钟**（每格 60 秒精耕细作）的创意故事短片。

## 4. 技术链路
- **前端渲染**: HTML5 Canvas + MediaRecorder。
- **后端支持**: Node.js (Express) 处理多媒体资产。
- **AI 赋能**: 对接图像生成与脚本优化模型，实现“创意即生成”。
